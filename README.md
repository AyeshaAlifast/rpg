# RPG Browser Game

## Overview

This is a simple browser-based RPG game built using HTML, CSS, and JavaScript. The project was created as a practice project to understand basic game logic, DOM manipulation, button-based interactions, and frontend game development.

The game is based on a dragon-fighting RPG concept where the player can visit different locations, fight monsters, earn gold, gain XP, and progress toward defeating the dragon.

This project was implemented for learning purposes by following a YouTube tutorial.

## Features

- Browser-based RPG game
- Player stats including XP, health, and gold
- Basic monster interaction
- Location-based game buttons
- Dragon fight option
- JavaScript-based game logic
- DOM manipulation practice
- Simple HTML and CSS structure

## Tech Stack

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html`.
4. The game will open directly in your browser.

No installation is required.
